SPRINT PRO — VERSION PWA PRÊTE À PUBLIER

Fichiers à envoyer sur GitHub :
- index.html
- manifest.webmanifest
- service-worker.js
- icon-192.png
- icon-512.png

Important :
1. Décompressez le ZIP.
2. Envoyez les 5 fichiers dans la racine du dépôt GitHub.
3. N’envoyez pas le ZIP lui-même.
4. Attendez quelques minutes, puis rechargez le site.
5. Sur iPhone : Safari > Partager > Sur l’écran d’accueil.
6. Sur Android : Chrome > menu > Installer l’application.

Le Worker FFA est déjà configuré dans index.html :
https://sprint-pro-ffa.maxime-afanou.workers.dev
